AWS Security + GenAI + Prowler

Agentic GenAI Security Accelerator

A Prowler-first AWS security posture accelerator that helps customers understand their current cloud security maturity, receive a 0–5 posture score, explore findings through a dashboard, and use AI to identify what will improve their score and strengthen their environment.

Overview

What We’re Building

The Problem

Customers often have fragmented security findings, little clarity on their true security posture, and no simple way to understand what actions will improve maturity the fastest.

The Solution

This accelerator uses Prowler to scan AWS environments, exports findings into Amazon S3, normalizes and scores them, and then uses Amazon Bedrock, MCP, and a customer-facing dashboard to turn findings into actionable, AI-guided security improvement.

Architecture

End-to-End Flow

The accelerator is designed as a Prowler-first pipeline with AI enrichment, posture scoring, and dashboard-based exploration.

Prowler

Performs AWS posture scanning and produces raw findings related to S3, IAM, CloudTrail, GuardDuty coverage, encryption, compliance gaps, and other security best practices.

S3 + Ingestion

Stores raw Prowler outputs as the findings lake. Lambda normalizes the findings into a common schema for scoring, analysis, and dashboarding.

AI Layer

Bedrock agents explain business impact, prioritize remediation, generate improvement roadmaps, and power a natural-language chat experience.

Capabilities

Core Features

Security Posture Scoring

Calculates a 0–5 maturity score across security pillars, helping customers understand where they are and what is missing.

Prowler Findings Explorer

Displays normalized Prowler findings by severity, service, resource, compliance mapping, and score impact.

AI Security Chat

Lets users ask questions like “Why is my score 2.7?” or “What will bring me to a 5?”

Remediation Guidance

Generates technical next steps, business impact explanations, and recommended remediation actions in dry-run mode.

MCP Validation

Uses MCP to validate guidance against AWS documentation and approved AWS context in a governed, explainable way.

Customer Dashboard

Gives customers an executive overview, findings explorer, score improvement view, remediation center, and reporting experience.

Scoring Model

0–5 Security Maturity Score

How the Score Works

  • Each pillar is scored from 0 to 5.
  • Controls are marked as Implemented, Partial, Missing, Unknown, or Not Applicable.
  • Not Applicable controls do not reduce the score.
  • Applicable missing controls reduce the score.
  • The dashboard shows what actions will increase the score the most.

  • Human approval required
  • CloudWatch audit logging
