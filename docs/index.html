Agentic GenAI Security Accelerator

A Prowler-powered AWS security posture accelerator that helps customers understand, score, and improve their cloud security maturity using AI, MCP, and dashboarding.

What It Does

  • Uses Prowler to scan AWS accounts for security posture findings.
  • Exports Prowler findings into Amazon S3 as the raw findings lake.
  • Normalizes findings for dashboarding and reporting.
  • Calculates a 0–5 security maturity score.
  • Uses Amazon Bedrock agents to explain findings and recommend remediation.
  • Uses MCP to validate AWS documentation and approved context.
  • Provides a customer-facing dashboard and AI security chat experience.

Architecture

Customer AWS Account → Prowler Scan → S3 Findings Lake → Ingestion Lambda → DynamoDB → Scoring Engine → Bedrock Agents → MCP Validation → Dashboard + AI Chat + Reports

Dashboard Capabilities

  • Executive security posture overview
  • Prowler findings explorer
  • 0–5 maturity score by pillar
  • Score improvement recommendations
  • AI security chat
  • Remediation center
  • Reports and audit history

Example Questions

  • Why is my score 2.7?
  • What did Prowler find?
  • What will bring me to a 5?
  • Do I need AWS WAF?
  • Which S3 buckets are exposed?
  • What should I fix first?

Governance

  • Read-only by default
  • Dry-run remediation
  • Human approval required
  • CloudWatch audit logging
  • S3 and DynamoDB encryption
  • Least-privilege IAM
